# How to start

## Node version

- see `.nvmrc`
- run `nvm use`
- if the node version is not installed, run `nvm install`

## Environment variable

- create `.env` and set `VITE_API_URL=REPLACE_THIS_WITH_SERVER_URL`

## Install dependencies

- run `npm i`

## Run the application

- run `npm run dev`

## Check lint

- run `npm run lint`
